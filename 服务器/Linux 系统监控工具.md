1.vmstat - 虚拟内存统计

vmstat 命令报告有关进程、内存、分页、块 IO、中断和 CPU 活动等信息。


```
# vmstat 3
```

2.找出占用内存资源最多的前 10 个进程

```
ps -auxf | sort -nr -k 4 | head -10
```

3.找出占用 CPU 资源最多的前 10 个进程


```
ps -auxf | sort -nr -k 3 | head -10
```
